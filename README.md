# Dojo Website Project 1

Fresh Next.js App Router build for the ENIRIQ Version D website.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser. If that port is already busy, run:

```bash
npm run dev -- -p 3001
```

## Validation

```bash
npm run lint
npm run build
```

## Notes

- Visible copy and placeholder data live in `src/content.ts`.
- Build handover and technical records live in `docs/`.
- The home hero uses the optimized particle canvas background in `src/components/fx/ParticleHeroBackground.tsx`.
